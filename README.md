# Life Chart Builder

あなたの人生曲線（幸福度・モチベーション曲線）を美しく描くWebツールです。ワークショップ等での活用を想定しています。

## 特徴

- **🔒 サーバー通信なし・完全ローカル**: サーバーへの通信を一切行いません。入力されたデータはすべてお使いのブラウザ内（LocalStorage）にのみ保存されるため、プライバシーが守られます。
- **💾 データの保存と移行**: データをJSON形式でエクスポート・インポートできるため、別のパソコンやブラウザへ作業を引き継ぐことができます。CSV形式での出力にも対応しています。
- **📄 A4横に最適化した画像保存**: スマホやPCの画面サイズに関わらず、保存される画像は常にA4横の比率（高解像度）で、1枚に綺麗に収まった状態で出力されます。
- **📱 スマホ対応（レスポンシブ）**: 画面幅が狭いスマホでは縦並びのレイアウトに自動調整されます。また、イベントが多くて密集する場合は自動的に横スクロール化し、見やすさを保ちます。

## ファイル構成

- `index.html`: メインの画面構造
- `style.css`: スタイルシート（レスポンシブ対応含む）
- `app.js`: グラフ描画やデータ管理のロジック

## 使い方

1. `index.html` をブラウザで開きます。
2. 生年月日を入力して「チャートを作成する」をクリックします。
3. 自動生成されたライフイベントの幸福度をスライダーで調整したり、新しいイベントを追加・編集します。
4. 完成したら「画像保存」でPNG画像としてダウンロードしたり、データをJSONでバックアップしてください。

## 開発・カスタマイズ

ソースコードがHTML/CSS/JSに分離されているため、以下のファイルを用途に合わせて編集してください：
- デザインの変更：`style.css`
- ロジック・計算の変更：`app.js`
